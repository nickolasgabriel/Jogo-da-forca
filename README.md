# JOGO DA FORCA
Este projeto consiste no desenvolvimento de um jogo da forca para dois jogadores, utilizando a linguagem Java e apenas conceitos básicos de programação. No jogo, o primeiro jogador escolhe uma palavra secreta, enquanto o segundo jogador tenta adivinhá-la digitando uma letra por vez.

O programa exibe a palavra oculta com traços (_) e, a cada tentativa, verifica se a letra informada existe na palavra. Caso exista, todas as ocorrências da letra são reveladas; caso contrário, o jogador perde uma tentativa. O jogo continua até que o jogador descubra toda a palavra ou atinja o número máximo de erros permitidos.

O desenvolvimento utiliza apenas conhecimentos básicos da linguagem Java, como:

Arrays para armazenar as letras da palavra e as letras descobertas;
ArrayList para guardar as letras que já foram digitadas pelo jogador;
Estruturas de repetição (for, while e do-while) para controlar o fluxo do jogo;
Estruturas condicionais (if, else) para verificar as jogadas e definir as ações do programa.

O objetivo do projeto é colocar em prática os principais conceitos de lógica de programação e programação orientada à resolução de problemas, desenvolvendo um jogo simples, funcional e de fácil compreensão.
